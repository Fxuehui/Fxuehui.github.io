身份（需要操作谁的权限）
u - 拥有文件的用户（所有者）
g - 所有者所在的族群(同组用户)
o - 其他人（不是所有者或所有者的组群）
a - 每个人或全部（u、g和o）
权限（需要修改的全新值）
r - 读取权
w - 写入权
x - 执行权
行动（实际的操作）
+ - 添加权限
- - 删除权限
= - 使它成为唯一权限

示例：
chmod o+w FileName //给其他用户添加可写的权限
chmod u-x FileName //去掉所属用户的可执行权限
chmod g=wx FileName //修改文件的同组用户的权限为可写以及可执行

1 chmod 367 Even // 修改文件Even的权限为
拥有者 ——可写+可执行 //-wx 3
同组用户——可读+可写 //rw- 6
其它用户——可读+可写+可执行 //rwx 7

**设置某个环境变量**

1. 临时设置（以PATH为例）：

gec@ubuntu:~$ export PATH=$PATH:/home/gec

注意，以上设置只对当前的终端有效。

1. 永久设置（以PATH为例）：

a. 打开终端的配置文件 

vim  ~/.bashrc

b. 在其末尾添加如下信息：

export PATH=$PATH:/home/gec export PATH=/home/gec:$PATH export   //设置临时的环境变量 PATH    // 需要设置的环境变量为  PATH =        // 赋值符号（左右两边不要留空格） $PATH     // $引用一个变量  $PATH 引用原本的path （保留原本的值【追加新的值】） :            // 分隔开多个不同的值 /home/gec  // 添加新的值

注意：需要执行如下命令来加载修改过的配置文件：

gec@ubuntu:~$ source ~/.bashrc

VIM编辑器默认有3种：

命令模式    		编辑模式 			尾行模式 （在命令前面增加 ：） 

以上模式之间如何切换：

默认打开处于命令模式，输入 i\a\o\O 则会进入编辑模式， 在编辑模式下就可以任意对文件进行编辑，按下 ESC 按键则可以退出编辑模式。在命令模式下输入 ： 冒号 则进入尾行模式 。 （保存并退出文件可以直接输入命令 ZZ ）

在使用vim 来打开某一个文件的时候，默认处于命令模式   ， 然后可以从键盘中输入命令进行执行操作：

**vim常用指令**

| vim命令     | 含义                                                         |
| ----------- | ------------------------------------------------------------ |
| :w          | 保存当前文件                                                 |
| :w filename | 另存为 filename （:w /home/even/Desktop/stdio.h）            |
| :q          | 退出当前正在编辑的文件（如果刚才有对文件进行修改则无法退出） |
| :q!         | 强制退出当前正在编辑的文件并放弃最近一次(不保存)保存到现在的所有操作 |
| :wq         | 保存文件并退出                                               |
| :%s/old/new | 将文件中所有的old字符串替换成new                             |
| i           | 在光标所在的位置前面插入字符                                 |
| a           | 在光标所在的位置后面插入字符                                 |
| o           | 在光标所在行的下一行插入新的一行                             |
| O           | 在光标所在行的上一行插入新的一行                             |
| u           | 撤销最近一次操作。（按Ctrl+r恢复撤销掉的操作）               |
| x           | 剪切光标处所在的字符。（x前可先按一个数字，则剪切若干个字符） |
| dd          | 剪切光标处所在的一行。（dd前可先按一个数字，则剪切若干行）   |
| yy          | 复制光标处所在的一行。（yy前可先按一个数字，则复制若干行）   |
| d$          | 剪切从光标处（含）开始到该行行末的所有字符                   |
| d^          | 剪切从光标处（不含）开始到该行行首的所有字符                 |
| y$          | 复制从光标处（含）开始到该行行末的所有字符                   |
| y^          | 复制从光标处（不含）开始到该行行首的所有字符                 |
| p           | 将剪切板中的资料粘贴到光标所在处                             |
| r           | 修改光标所在的字符，r之后接跟你要修正的字符（比如你要把fox中的o改成i，只需将光标停在o上，接着连续按下r和i） |
| h           | 将光标向前（左）移动一个字符                                 |
| j           | 将光标向下移动一个字符                                       |
| k           | 将光标向上移动一个字符                                       |
| l（小写L）  | 将光标向后（右）移动一个字符                                 |
| gg          | 跳到文本的最初一（第一）行                                   |
| G           | 跳到文本的最末一行                                           |
| Ctrl + u    | 向上(up)翻页                                                 |
| Ctrl + d    | 向下(down)翻页                                               |
| /string     | 从光标处往下查找字符串string，注意在输完你要查找的字符串string之后要按回车键。如果你要找的字符串string有多个，你可以按n将光标跳到下一个位置，按N将光标跳到上一个位置 |
| ?string     | 跟上面的是一样的，区别是它从光标处往上查找                   |
