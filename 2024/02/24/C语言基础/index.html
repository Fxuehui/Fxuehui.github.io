安装

gec@ubuntu:~$ sudo apt install xxx

卸载

gec@ubuntu:$ sudo apt remove xxx
gec@ubuntu:$ sudo apt remove –purge xxx #将配置文件也一并删除

修复APT管理器

gec@ubuntu:~$ sudo apt install -f

自动卸载一些已经无用的软件或服务

gec@ubuntu:~$ sudo apt autoremove

如果安装过程中提示缺少相关依赖，则执行如下命令解决：
sudo apt -f install
